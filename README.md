# Student Performance Analyzer (AI)

This project analyzes and predicts student performance based on study hours using Machine Learning.

## Features

* Linear Regression model
* Data visualization
* Prediction system

## Technologies

* Python
* scikit-learn
* matplotlib

## How it works

The model is trained on sample data and predicts student scores based on study time. A graph is generated to visualize the relationship.
